# FTB-Sky-Odyssey-Translation

## 魔改内容 

增加了3个模组：
 1. i18n自动下载翻译：用于汉化未支持中文的模组，下载后自动启用。
 2. 模块化组装：塔迪斯发电机几千个方块！太难摆！加入这个模组可以自动建造。具体内容请看游戏内物品栏左上角的指南。
 3. 通用拼音搜索：这个模组可以用拼音搜索JEI等输入框的内容。
